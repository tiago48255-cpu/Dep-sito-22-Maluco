# 📥 Coloque aqui o design do Stitch

Esta pasta é onde o Tiago joga o design exportado do Stitch.
Quando estiver tudo aqui, é só falar **"tá lá"** no chat.

## O que colocar

### 1. Os arquivos HTML (o design em si)
- No Stitch: menu **Exportar → `.zip`** (baixa todas as telas de uma vez)
- Descompacte o `.zip`
- Copie os arquivos `.html` pra dentro desta pasta
- Um arquivo por tela. Ex: `home.html`, `categorias.html`, `checkout.html`

> Alternativa: pra mandar UMA tela por vez, use
> **Exportar → "Código para a área de transferência"** e cole o código no chat.

### 2. Os prints (pra validar a fidelidade)
- Um print da versão **celular** e um da versão **desktop** de cada tela
- Nomeie pela tela + dispositivo. Ex:
  - `home-mobile.png` / `home-desktop.png`
  - `checkout-mobile.png` / `checkout-desktop.png`

## ⚠️ NÃO use estas opções do Stitch (não são o design)
AI Studio · Figma · MCP · Netlify · Lovable · Bolt · "Resumo do projeto"
